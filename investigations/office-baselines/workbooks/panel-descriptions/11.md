### 👤 vs 🌐 11. One person against the organisation over time

| Workbook context | Value |
|---|---|
| Selected period | `{W_TimeRange:start}` to `{W_TimeRange:end}` |
| Person shown | `{W_TargetUser}` *(set this to choose the person)* |

This chart draws one person's daily activity and the organisation's average on the same chart. The gap between the two lines shows, day by day, whether the person is running above or below the typical level. Set the user filter to choose the person.

This is supporting context, not a league table.

#### What the chart shows

| Element | What it means |
|---|---|
| Horizontal axis (`Day`) | The day. |
| Vertical axis (`Value`) | The number of activity events for that day. |
| `Series` = `User` | The selected person's daily activity. |
| `Series` = `Org avg / active user` | The organisation's average activity per active person. |

#### What is included

Everyday staff activity only. Guests, system accounts and listed service accounts are excluded. If the user filter is blank, only the organisation line appears.
